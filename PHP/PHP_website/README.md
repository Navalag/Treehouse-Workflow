## Build a Basic PHP Website

### About this Course/Project:

This project is to build a simple website using the PHP programming language. The web site, a media library, will let you organize and display your Books, Movies and Music. You could easily use this project to create a website that lets you organize and display any type of item: Branch Locations, Products, Services, Events, etc.

### Skills:

- Organizing and Displaying Inventory
- Reusing Code
- Email and Form Data
